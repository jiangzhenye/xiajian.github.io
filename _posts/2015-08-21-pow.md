地址: http://pow.cx/

安装: 

```
curl get.pow.cx | sh
```

使用: 

$ cd ~/.pow
$ ln -s /path/to/myapp

配置: 

```
# .powrc
if [ -f "$rvm_path/scripts/rvm" ] && [ -f ".rvmrc" ]; then
  source "$rvm_path/scripts/rvm"
  source .rvmrc
fi
```

创建 rvmrc：

```
rvm current # => 
rvm use ruby-2.1.5@shops-alto --create --rvmrc
# 信任 rvmrc
rvm rvmrc warning ignore all.rvmrcs
```
