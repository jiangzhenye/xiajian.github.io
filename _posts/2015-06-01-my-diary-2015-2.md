## 7月8日

修改了一个 js，发现数组自带了 join 函数，搞的自己还又写了一个，还用了一下 underscore 的库： 

```
function get_sku_key() {
  var sku_key = new Array();
  var tkey = "";
  $(".tb_sku_panel ul li.selected").each(function () {
    sku_key.push($(this).attr("data-id"));
  });
  _.map(sku_key, function (i) {
    if (tkey == "") {
      tkey = i
    } else {
      tkey = tkey + "_" + i;
    }
  })
  return tkey;
}
```
