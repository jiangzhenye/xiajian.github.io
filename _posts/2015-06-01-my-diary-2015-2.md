shell 提供的编程太恶心了，不如 Ruby 的好用，尤其是字符串和数组的操作，今天，做了一个将当前目录下所有文件软连接到另一个目录下: 

```ruby
a = `ls`
a.split("\n").each { |x| system "ln -s #{x}  ../../../config/#{x}" }  # 产生一个问题，相对路径有问题
a.split("\n").each { |x| system "ln -s #{x}  /Users/xiajian/works/fesco/config/#{x}" }  # 结果生成了 boken softlink
a = `ls /Users/xiajian/works/fesco/environments/luo.facloud.com/config`
a.split("\n").each do |x| 
  system "rm /Users/xiajian/works/fesco/config/#{x}"
  system "ln -sf  /Users/xiajian/works/fesco/environments/luo.facloud.com/config/#{x} #{x}"  # 软连接正坑爹
end
```

如果，能在 irb 中执行 shell 命令的话，配合 ruby 强大简洁的表述能力，一定可以舒服很多。

VIM 中添加了两个键映射，具体介绍如下。虽然没了当前选择区，但得到了 pbcopy 和 pbpaste 命令，有得必有失。

```
" 参考: http://vim.wikia.com/wiki/Mac_OS_X_clipboard_sharing 
" F7 粘贴，F8 复制，此时，已经将F1-F6 都映射完成了
nmap <F7> :set paste<CR>:r !pbpaste<CR>:set nopaste<CR>
imap <F7> <Esc>:set paste<CR>:r !pbpaste<CR>:set nopaste<CR>
nmap <F8> :w !pbcopy<CR><CR>
vmap <F8> :w !pbcopy<CR><CR>
```
