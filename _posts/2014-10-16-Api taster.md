## 简介
----
ApiTaster是一个简单的，可视的方法，可以用来测试Rails应用程序的API。

> 0.8版本的只能和Rails 4.1 一起使用，如果想要在Rails 3.x/4.0中使用，请指定0.7版本的Api Taster。

## 为何

世界上已经了一坨API客户端（比如，Postman), 所以，为何要重造轮子。

API Taster拥有如下的这些特性可选: 

* API终端点是自动从Rails路由的定义中生成的
* 定义参数和定义路由一样的方便
* 参数可以在测试工厂之间共享
